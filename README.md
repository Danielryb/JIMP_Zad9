# Treść zadania
## Zad9
Proszę przeanalizować strukturę zaproponowanego projektu, który rozwiązuje macierzowe układy równań liniowych metodą eliminacji Gaussa z wyborem elementu głównego, a następnie dopisać odpowiednie fragmenty kodu:
- eliminate – implementująca schodkowanie macierzy z wyborem elementu głównego,
- backsubst – wykonująca wsteczne podstawienie.

### Co będzie oceniane?
- 2pkt – merge gałęzi rozwojowych
- 4pkt – schodkowanie + wsteczne podstawienie
- 2pkt - testy
- 2pkt – wybór elementu głównego

MAX 10pkt

wikidyd: http://wikidyd.iem.pw.edu.pl/LMP(2f)9.html

- metoda gradientów sprzężonych (zamiast rozwiązywania układu równań metodą Gaussa)
